Expectation Failed | Error 417
!

기대값이 실패했습니다

Date:

왜 이런 오류가 발생했나요?

요청의 Expect 헤더 필드에 있는 기대값을 서버가 만족할 수 없습니다.

The server cannot meet the requirements of the Expect request-header field.

주로 "Expect: 100-continue" 헤더를 서버가 처리할 수 없을 때 발생합니다.

This typically occurs when the server cannot handle the "Expect: 100-continue" header.

어떻게 해결할 수 있나요?

Expect 헤더 없이 요청을 다시 시도하거나 다른 방법을 사용해 주세요.

Please retry the request without the Expect header or use an alternative method.

문제가 지속되면 웹사이트 관리자에게 문의해 주세요.

If the problem persists, please contact the website administrator.
